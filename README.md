# Analizador de Manuales de Instrucciones

Este proyecto es una herramienta de la **Dirección Técnica IA LAB** diseñada para extraer y analizar automáticamente la información preventiva de manuales de maquinaria y equipos de trabajo.

## Funcionalidad Principal
- **Extracción de Texto**: Procesamiento de PDFs.
- **Análisis IA**: Identificación automática de riesgos, EPIs y medidas de seguridad usando LLMs.
- **Interfaz**: Visualización estructurada de los hallazgos.

## Configuración
El nombre interno del proyecto es `direccion-tecnica-manuals`.
